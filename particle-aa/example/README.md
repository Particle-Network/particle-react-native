# particle-aa Example

This example app demonstrates the `@particle-network/rn-aa` SDK, covering Account Abstraction features such as deploying smart accounts, sending user operations, and gas sponsorship.

## Setup

```sh
npm install
```

## Run

### Android

```sh
npm run android
```

### iOS

```sh
npm run ios
```

> **Note:** iOS builds require a physical device (`ios-arm64`). Simulators are not supported by the underlying SDK.

For more information, see the [Particle AA React Native docs](https://developers.particle.network/api-reference/aa/sdks/mobile/react).
