# particle-base Example

This example app demonstrates the `@particle-network/rn-base` package and the shared configuration used across Particle Network's React Native SDKs.

## Setup

```sh
npm install
```

## Run

### Android

```sh
npm run android
```

### iOS

```sh
npm run ios
```

For more information, see the [Particle Network React Native docs](https://developers.particle.network/api-reference/auth/mobile-sdks/react).
