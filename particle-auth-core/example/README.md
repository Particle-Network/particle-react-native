# particle-auth-core Example

This example app demonstrates the `@particle-network/rn-auth-core` SDK, covering social login flows and MPC-based wallet creation.

## Setup

```sh
npm install
```

## Run

### Android

```sh
npm run android
```

### iOS

```sh
npm run ios
```

> **Note:** iOS builds require a physical device (`ios-arm64`). Simulators are not supported by the underlying SDK.

For more information, see the [Particle Auth Core React Native docs](https://developers.particle.network/api-reference/auth/mobile-sdks/react).
