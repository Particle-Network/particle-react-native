# particle-connect Example

This example app demonstrates the `@particle-network/rn-connect` SDK, covering the unified connection flow with social logins and external wallet integrations, chain selection, and message signing.

## Setup

```sh
npm install
```

## Run

### Android

```sh
npm run android
```

### iOS

```sh
npm run ios
```

> **Note:** iOS builds require a physical device (`ios-arm64`). Simulators are not supported by the underlying SDK.

For more information, see the [Particle Connect React Native docs](https://developers.particle.network/api-reference/connect/mobile/react).
