# particle-wallet Example

This example app demonstrates the `@particle-network/rn-wallet` SDK, covering the built-in wallet UI and wallet management APIs.

## Setup

```sh
npm install
```

## Run

### Android

```sh
npm run android
```

### iOS

```sh
npm run ios
```

> **Note:** iOS builds require a physical device (`ios-arm64`). Simulators are not supported by the underlying SDK.

For more information, see the [Particle Wallet React Native docs](https://developers.particle.network/api-reference/wallet/mobile/react).
