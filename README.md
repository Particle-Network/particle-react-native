> **Note for iOS:** `particle_auth_core` only supports `ios-arm64` (physical devices). Simulators are not supported — you will need an actual iPhone for testing.

## Packages

| Package | Install |
|---|---|
| Auth Core | `npm install @particle-network/rn-auth-core` |
| Connect | `npm install @particle-network/rn-connect` |
| Wallet | `npm install @particle-network/rn-wallet` |
| Account Abstraction | `npm install @particle-network/rn-aa` |

## Getting Started

1. Run the example app inside `particle-connect/example`
2. Select a chain
3. Connect with a social login or wallet
4. Explore message signing and other features

### Expo

If you are using Expo, an example project is available under `Expo-Examples/new-expo-app`. For iOS development, update your Podfile by following the [Expo iOS Podfile](https://github.com/Particle-Network/particle-react-native/blob/master/Expo-Examples/new-expo-app/ios/Podfile) reference.
